## Dependency
- Gradle
```
repositories {
    maven {
       name = 'FastMCMirror Repo'
       url = 'https://repo.fastmcmirror.org/content/repositories/releases/'
    }
}
dependencies {
    implementation 'org.fastmcmirror:InternationalizationAPI:${version}'
}
```
- ${version} Replace with the latest version
